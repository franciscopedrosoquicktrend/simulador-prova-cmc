# Áudio do examinador

A amostra fornecida está guardada em:

- `siri-voice-demo.m4a`

É utilizada apenas no botão de demonstração da voz.

Coloque nesta pasta as gravações finais:

- `m32-examiner.m4a`
- `etc-examiner.m4a`

Quando estes ficheiros existirem, o simulador reproduz as gravações em vez da voz de síntese disponibilizada pelo navegador.

Amostras curtas de voz não permitem gerar texto novo. Cada instrução deve ser exportada como um ficheiro de áudio completo.
